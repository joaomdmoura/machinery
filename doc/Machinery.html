    machinery v0.2.0 Machinery View Source

    Main Machinery module. It keeps the bunk of the Machinery logics, it’s the module that will be imported with use on the module that the state machine will be implemented.

    Parameters

    • opts: A Keyword including states and transitions.

    Example 

      defmodule Project.User do
    use Machinery,
      states: [:created, :partial, :complete],
      transitions: [
        %{from: :created, to: [:partial, :complete]},
        %{from: :partal, to: :complete}
      ]
  end
